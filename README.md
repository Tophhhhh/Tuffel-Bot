# Commands

## Slash-Commands
| Command | Description | Rights needed |
|------|--------------------------|--------|
| /moveall | move all member of your channel to an other vc | Move |
| /coinflip | get value `Head` or `tail` | - |
| /weather | get weather of a city | - |
| /poll [Question]| create poll | - |
